# 컴포넌트: 빈 상태 (Empty State)

## 개요
섹션에 콘텐츠가 없을 때 표시. 다음 액션으로 사용자를 안내.

---

## 디자인 토큰

| 토큰 | 값 |
|---|---|
| 패딩 | 48px 24px |
| 아이콘 크기 | 40px (이모지 또는 일러스트) |
| 아이콘 불투명도 | 0.5 |
| 제목 | `text/heading-lg` = 18px / SemiBold |
| 설명 | `text/body-md` = 16px / Regular, `color/text/secondary` |
| 설명 최대 너비 | 320px |
| CTA 버튼 | Primary 또는 Secondary 버튼 |

---

## 구조 (중앙 정렬, 세로 쌓기)
1. 아이콘 / 일러스트 (40px, 50% 불투명도)
2. 간격: 16px
3. 제목 (18px / SemiBold)
4. 간격: 8px
5. 설명 텍스트 (중앙 정렬, 최대 너비 320px)
6. 간격: 20px
7. CTA 버튼 (선택)

---

## 변형

### 데이터 없음
- 아이콘: 관련 이모지 (📋, 📁 등)
- 제목: "아직 [항목]이 없어요"
- 설명: 도움이 되는 안내 텍스트
- CTA: 생성/추가를 위한 Primary 버튼

### 검색 결과 없음
- 아이콘: 🔍
- 제목: "검색 결과가 없어요"
- 설명: 다른 검색어 제안
- CTA: 검색 초기화를 위한 Secondary 버튼

---

## Figma Make 프롬프트

```
다음 스펙으로 빈 상태(Empty State) 컴포넌트를 만들어줘:

레이아웃: 세로 중앙 정렬, 모든 요소 가운데 정렬
패딩: 상하 48px, 좌우 24px

구조:
1. 대형 아이콘/이모지 (40px, 약간 흐린 불투명도)
2. 제목 (18px SemiBold, 아이콘 아래 16px 간격)
3. 설명 텍스트 (16px Regular, 회색, 최대 너비 320px, 제목 아래 8px)
4. 선택적 CTA 버튼 (설명 아래 20px)

변형:
- 데이터 없음: 일반 빈 클립보드 아이콘, "아직 항목이 없어요"
- 결과 없음: 검색 아이콘, "검색 결과가 없어요"

네이밍: Empty State / No Data, Empty State / No Results
```

# 컴포넌트: 셀렉트 / 드롭다운 (Select)

## 개요
단일 옵션 선택기. 우측에 드롭다운 쉐브론이 있는 스타일 인풋으로 표시.

---

## 디자인 토큰

인풋과 동일한 기본 토큰.

| 토큰 | 값 |
|---|---|
| 쉐브론 아이콘 | 16px, `color/text/tertiary`, 우측 정렬 |
| 쉐브론 패딩 | 텍스트 겹침 방지를 위한 우측 패딩 36px |
| Appearance | none (브라우저 기본 스타일 제거) |

---

## 상태
기본, 호버, 포커스, 비활성 — 인풋과 동일.

---

## Figma Make 프롬프트

```
다음 스펙으로 셀렉트/드롭다운(Select) 컴포넌트를 만들어줘:

인풋 컴포넌트와 동일한 스타일에:
- 우측에 쉐브론 화살표 아이콘 (16px)
- 텍스트/아이콘 겹침 방지를 위한 추가 우측 패딩 (36px)
- 브라우저 기본 쉐브론 제거
- 패딩: 9px 36px 9px 12px

상태: 기본, 호버, 포커스 (오렌지 보더 + 링), 비활성

네이밍: Select / {상태}
```

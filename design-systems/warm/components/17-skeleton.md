# 컴포넌트: 스켈레톤 (Skeleton)

## 개요
콘텐츠 로딩 중 표시되는 플레이스홀더 UI. 레이아웃 이동 방지.

---

## 디자인 토큰

| 토큰 | 값 |
|---|---|
| 기본 색상 | `color/surface/3` |
| 시머 색상 | `color/border/default` |
| 애니메이션 | shimmer 1.5s infinite (배경 위치 스윕) |
| 텍스트 반경 | 6px |
| 원형 반경 | 50% |
| 사각형 반경 | `radius/card` = 12px |

---

## 요소 종류

### 텍스트 스켈레톤
- 높이: 12px (sm), 14px (기본), 18px (lg)
- 보더 반경: 6px
- 너비: 다양 (100%, 85%, 65% 등)
- 하단 여백: 줄 사이 8px

### 원형 스켈레톤
- 아바타 자리 표시자로 사용
- 보더 반경: 50%
- 일반 크기: 36px, 44px

### 사각형 스켈레톤
- 이미지, 카드, 미디어 자리 표시자
- 보더 반경: `radius/card` (12px)
- 가변 가로/세로

---

## 패턴

### 카드 스켈레톤
- 헤더 행: 원형 (아바타) + 텍스트 두 줄
- 대형 사각형 (이미지/미디어 영역)
- 하단 텍스트 세 줄

### 목록 스켈레톤
- 반복 행: 원형 + 텍스트 두 줄 + 짧은 우측 텍스트

---

## Figma Make 프롬프트

```
다음 스펙으로 스켈레톤(Skeleton) 로딩 플레이스홀더 컴포넌트를 만들어줘:

기본 스타일: 시머 애니메이션이 있는 연한 회색 채움 (좌→우 스윕)

요소:
- 텍스트 스켈레톤: 둥근 사각형, 높이 12px/14px/18px
- 원형 스켈레톤: 완전한 원형 (아바타 자리 표시자)
- 사각형 스켈레톤: 둥근 사각형 (이미지/카드 자리 표시자)

패턴:
- 카드 스켈레톤: 아바타 행 + 이미지 영역 + 텍스트 3줄
- 목록 스켈레톤: 아바타 + 텍스트 2줄 + 짧은 우측 텍스트로 구성된 3개 행

모든 요소에 동일한 회색 시머 애니메이션 적용

네이밍: Skeleton / Text / {크기}, Skeleton / Card, Skeleton / List
```

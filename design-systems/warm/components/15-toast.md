# 컴포넌트: 토스트 / 스낵바 (Toast)

## 개요
자동으로 나타났다가 사라지는 임시 플로팅 알림.

---

## 디자인 토큰

| 토큰 | 값 |
|---|---|
| 배경 | `color/text/primary` (반전) |
| 텍스트 | `color/surface/1` (반전) |
| 보더 반경 | `radius/card` = 12px |
| 패딩 | 12px 16px |
| 그림자 | `shadow/toast` |
| 최대 너비 | 360px |
| 자동 닫힘 | 3초 |
| 등장 모션 | `motion/scale` = 200ms spring |
| 퇴장 모션 | `motion/fade` = 200ms ease-out |

---

## 구조
- 아이콘 (15px) — 좌측
- 메시지 텍스트 (13px / Medium) — flex-1
- 선택적 액션 버튼 (12px / SemiBold, primary-tint 색상)
- 닫기 버튼 (✕, 14px) — 우측

---

## 위치
- 고정: 우측 하단, 가장자리에서 24px
- 여러 토스트 스택: 8px 간격으로 세로 쌓기

---

## Figma Make 프롬프트

```
다음 스펙으로 토스트(Toast) 알림 컴포넌트를 만들어줘:

배경: 어두운 색상 (#1A1916 라이트 / #FAFAF8 다크) — 페이지 반전
텍스트: 흰색/밝은 색상 (13px Medium)
보더 반경: 12px
엘리베이션 드롭 섀도우
최대 너비: 360px

구조: 아이콘 + 메시지 + 선택적 액션 링크 + 닫기 (✕)

변형:
- 기본 (아이콘 포함)
- 액션 버튼 포함 (오렌지-틴트 텍스트 링크)

위치: 우측 하단 모서리
애니메이션: 스프링 팝인, 페이드 아웃

네이밍: Toast / Default, Toast / With Action
```
